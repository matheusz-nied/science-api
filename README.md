# Nied Science API

Nied Science API é um projeto desenvolvido em Go, que serve como backend para um aplicativo de ciência. Ele utiliza a API da NASA para fornecer dados astronômicos diários.

## Tecnologias Utilizadas

- [Golang](https://golang.org/)
- [Gin Gonic](https://github.com/gin-gonic/gin)
- [GORM](https://gorm.io/)
- [PostgreSQL](https://www.postgresql.org/)
- [Docker](https://www.docker.com/)

## Configuração do Projeto

### Pré-requisitos
